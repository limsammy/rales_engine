# How to Setup

This readme will walk you through the setup and installation of this app.

### Running Locally

  * Clone this repo.
  * cd into it
  * ```bundle install```
  * ```rake db:create```
  * ```rake db:migrate ```
  * ```rake import_csv:all```
  * ```rails server```
  * ```Visit localhost:3000```

### Running the test suite
  * Run ```rspec``` from the root directory
